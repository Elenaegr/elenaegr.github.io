The Lyric Searching App

The lyric searching app is the project resulting from the Interaction Programming and the Dynamic Web course of the Interactive Media Technology Master’s programme at KTH Royal Institute of Technology. The lyric searching app is a functional live app where the user can look for the lyrics of its favorite songs, look for the lyrics of its favorite albums, save them on its profile and translate them into several languages.

The app is built using React along with several different modules: the Musixmatch API to render the lyrics just to a certain extent since we used the free version of the API, the Google Translate API to translate the lyrics to a total of 103 different languages and the Firebase services such as authentication to allow the saving features for users, database to store users’ access information and hosting to host our Lyric searching site on the Internet.
